01

Overview 概述

Depending on the model and provider you choose, you can adjust numerous parameters that define the model's output and connectivity.

EN

Two Categories of Parameters

Model Output — the level of creativity or determinism in the generated content (text, images), the volume of content generated, etc.

Connectivity — base URL, authorization keys, timeouts, retries, logging, etc.

Typically, you will find all the parameters and their meaning on the model provider's website. For example, OpenAI API's parameters can be found at platform.openai.com/docs/api-reference/chat.

中文

两类参数

模型输出 — 生成内容（文本、图像）的创造力或确定性程度、生成内容的数量等。

连接性 — 基础 URL、授权密钥、超时、重试、日志记录等。

通常，你可以在模型提供商的网站上找到所有参数及其含义。例如，OpenAI API 的参数可在 platform.openai.com/docs/api-reference/chat 查阅。

Common Parameters 常用参数

Key OpenAI API parameters. For the full list, see the OpenAI Language Model page. Full lists per model can be found under Integration, Language Model and Image Model pages.

Parameter / 参数 Description / 描述 Type / 类型
modelName The name of the model to use (e.g., gpt-4o, gpt-4o-mini, etc.)
要使用的模型名称		 String
temperature What sampling temperature to use, between 0 and 2. Higher values like 0.8 make output more random; lower values like 0.2 make it more focused and deterministic.
采样温度，0~2。值越高越随机，越低越聚焦和确定		 Double

03

Creation Methods 创建方式

You can create *Model in two ways: a static factory with sensible defaults, or a builder pattern for full control.

EN

Static Factory

A static factory that accepts only the mandatory parameters, such as API keys, with all other parameters set to sensible defaults.

Builder Pattern

A builder pattern: here, you can specify the value for each parameter individually, giving you full control over the model configuration.

中文

静态工厂

静态工厂方法仅接受必需参数（如 API 密钥），其余参数均设置为合理的默认值。

构建器模式

构建器模式：你可以为每个参数单独指定值，从而完全掌控模型配置。

04

Model Builder 模型构建器

We can set every available parameter of the model using the builder pattern as follows.

EN

OpenAiChatModel Builder

Set every available parameter of the model using the builder pattern. Parameters include API key, model name, temperature, timeout, and logging flags.

中文

OpenAiChatModel 构建器

使用构建器模式设置模型的每个可用参数，包括 API 密钥、模型名称、温度、超时和日志标志。

OpenAiChatModel.java 
OpenAiChatModel model = OpenAiChatModel.builder()
    .apiKey(System.getenv("OPENAI_API_KEY"))
    .modelName("gpt-4o-mini")
    .temperature(0.3)
    .timeout(ofSeconds(60))
    .logRequests(true)
    .logResponses(true)
    .build();

Setting Parameters in Quarkus 在 Quarkus 中设置参数

LangChain4j parameters in Quarkus applications can be set in the application.properties file.

Quarkus Config

application.properties

Configure model parameters directly in the properties file. The DEV UI dashboard allows live tweaking — changes are immediately reflected in your running instance and automatically ported to code.

在属性文件中直接配置模型参数。DEV UI 仪表板支持实时调整——更改会立即反映到运行实例并自动同步到代码。

Spring Boot Config

application.properties

If you are using one of our Spring Boot starters, configure model parameters in the properties file. The complete list of supported properties can be found in the official documentation.

若使用 Spring Boot 启动器，可在属性文件中配置模型参数。完整属性列表请参阅官方文档。

EN

Quarkus Configuration

Parameters can be set in application.properties. The DEV UI (accessible at localhost:8080/q/dev-ui when running quarkus dev) lets you tweak parameters live — changes reflect immediately and are auto-ported to code.

中文

Quarkus 配置

参数可在 application.properties 中设置。运行 quarkus dev 后，可通过 DEV UI（localhost:8080/q/dev-ui）实时调整参数——更改立即生效并自动同步到代码。

application.properties (Quarkus) 
# Quarkus LangChain4j Configuration
quarkus.langchain4j.openai.api-key=${OPENAI_API_KEY}
quarkus.langchain4j.openai.chat-model.temperature=0.5
quarkus.langchain4j.openai.timeout=60s
DEV UI — For debugging, tweaking, or exploring all available parameters, use the Quarkus DEV UI. Run quarkus dev and navigate to localhost:8080/q/dev-ui. Changes are immediately reflected and auto-ported to your code. 用于调试、微调或探索所有可用参数。运行 quarkus dev 并访问 localhost:8080/q/dev-ui，更改立即生效并自动同步到代码。
06

Setting Parameters in Spring Boot 在 Spring Boot 中设置参数

If you are using one of our Spring Boot starters, you can configure model parameters in the application.properties file.

EN

Spring Boot Configuration

Configure model parameters using the langchain4j.open-ai.chat-model.* property prefix in application.properties. The complete list of supported properties can be found in the official documentation.

中文

Spring Boot 配置

使用 langchain4j.open-ai.chat-model.* 属性前缀在 application.properties 中配置模型参数。完整属性列表请参阅官方文档。

application.properties (Spring Boot) 
# Spring Boot LangChain4j Configuration
langchain4j.open-ai.chat-model.api-key=${OPENAI_API_KEY}
langchain4j.open-ai.chat-model.model-name=gpt-4-1106-preview
...
More info — Quarkus integration details and Spring Boot integration details can be found in the official LangChain4j documentation. Quarkus 和 Spring Boot 集成的更多信息请参阅 LangChain4j 官方文档。
